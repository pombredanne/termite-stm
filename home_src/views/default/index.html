Data Server for Topic Models
Termite is a visualization tool for inspecting the output of statistical topic models. The tool contains two components: data servers and web-based visualizations.
This is a data server for processing the output of topic models (e.g., built using MALLET, ITM, STM, etc.) and distributing the content as a web service. For more information, see the Termite Data Server GitHub repository.
API Format
http:// [server] / [dataset] / [model] / [attribute]
By default, this server generates a human-readable HTML document (like this page!). To request a machine-readable response in JSON format, append ?format=json to any GET or POST actions.
